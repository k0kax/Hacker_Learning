# 1.Tag
##### 1.1创建tag
`git tag [标签名] -m [备注信息] [提交版本号]`
其中 **-m [备注信息]** 可以省略不写，但建议不要省略。**[提交版本号]** 可以省略，如果是给当前分支最新的提交创建标签的话
##### 1.2查看tag
粗略查看`git tag`
仔细查看 `git show 标签名`
##### 1.3删除tag
`git tag -d [标签名]`

##### 1.4将本地tag提交到远程仓库
`git push origin [标签名]`
